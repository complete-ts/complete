# `complete-node`

[![npm version](https://img.shields.io/npm/v/complete-node.svg)](https://www.npmjs.com/package/complete-node)

`complete-node` is a library that contains helper functions for a [Node.js](https://nodejs.org/) [TypeScript](https://www.typescriptlang.org/) project. (It can also be used in other runtimes, such as [Deno](https://deno.com/) and [Bun](https://bun.com/).)

You can browse the things that this library provides from the left navigation.
