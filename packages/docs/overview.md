# Overview

Complete is a collection of packages to make working with [TypeScript](https://www.typescriptlang.org/) easier.

## Packages

| Name                                                          | Description                                                | Version                                                                                                                          |
| ------------------------------------------------------------- | ---------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------- |
| [complete-cli](/complete-cli)                                 | A CLI tool to help you bootstrap a new TypeScript project. | [![](https://img.shields.io/npm/v/complete-cli.svg)](https://www.npmjs.com/package/complete-cli)                                 |
| [complete-common](/complete-common)                           | Helper functions for TypeScript projects.                  | [![](https://img.shields.io/npm/v/complete-common.svg)](https://www.npmjs.com/package/complete-common)                           |
| [complete-lint](/complete-lint)                               | A linting dependency meta-package for TypeScript projects. | [![](https://img.shields.io/npm/v/complete-lint.svg)](https://www.npmjs.com/package/complete-lint)                               |
| [complete-node](/complete-node)                               | Helper functions for Node.js projects.                     | [![](https://img.shields.io/npm/v/complete-node.svg)](https://www.npmjs.com/package/complete-node)                               |
| [complete-tsconfig](/complete-tsconfig)                       | A sharable TypeScript config for TypeScript projects.      | [![](https://img.shields.io/npm/v/complete-tsconfig.svg)](https://www.npmjs.com/package/complete-tsconfig)                       |
| [eslint-config-complete](/eslint-config-complete)             | A sharable ESLint config for TypeScript projects.          | [![](https://img.shields.io/npm/v/eslint-config-complete.svg)](https://www.npmjs.com/package/eslint-config-complete)             |
| [eslint-plugin-complete](/eslint-plugin-complete)             | An ESLint plugin that contains useful rules.               | [![](https://img.shields.io/npm/v/eslint-plugin-complete.svg)](https://www.npmjs.com/package/eslint-plugin-complete)             |
| [markdownlint-config-complete](/markdownlint-config-complete) | A sharable markdownlint config.                            | [![](https://img.shields.io/npm/v/markdownlint-config-complete.svg)](https://www.npmjs.com/package/markdownlint-config-complete) |
| [markdownlint-rule-complete](/markdownlint-rule-complete)     | A collection of opinionated custom rules for Markdownlint. | [![](https://img.shields.io/npm/v/markdownlint-rule-complete.svg)](https://www.npmjs.com/package/markdownlint-rule-complete)     |
